File download

In some cases it can be necessary to download a file in the device. For example, it can be a burst which needs to be installed on the device. For this purpose a downloader available in web-interface on page System/Upload file can be used.

Figure 5.10. File download


  • Path — a directory in file system to which the file will be downloaded;

  • File — a file on a local computer which needs to be downloaded.
