SNMP support level of the router

For SNMP support the router uses an unbound software package net-snmp , supporting all the actual versions of SNMP, which includes the snmpd daemon functioning as an agent.

The router supports MIB-2 which provides general information about the system (network interfaces, routing tables, system data), and HDSL2-SHDSL-LINE-MIB which includes the objects used for SHDSL channels monitoring and management.

MIB-2 is a standard MIB and is included in the majority of managers, while HDSL2-SHDSL-LINE-MIB is type-specific. This file can be downloaded from the Internet sources or by reference .

At the moment only monitoring of the router SNMP and SHDSL channels and regenerators is available, the management option will be added soon.
