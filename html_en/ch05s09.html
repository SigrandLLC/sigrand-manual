Configuration upload and failback

Web-interface allows saving of the current configuration or to back out saved or initial configuration of the router.

Configuration saving is performed on page System/Configuration/Backup:

Figure 5.7. Configuration backup


When clicking the "Backup" button there will be offered to save the configuration file, which can be loaded in future.

Important

After the failback the device will reboot automatically.

The failback is performed on page System/Configuration/Restore:

Figure 5.8. Failback


On the tap System/Configuration/default it is possible to back out the initial configuration by clicking the Restore default button.

Figure 5.9. Back-out of factory default model
